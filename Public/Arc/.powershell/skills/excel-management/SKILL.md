# Excel Management

This skill allows you to manage Excel files and perform various operations on them, such as importing and exporting data.

## Import Excel

```powershell
Import-Excel -Path "C:\path\to\file.xlsx"
```

## Export Excel

```powershell
Export-Excel -Path "C:\path\to\file.xlsx" -WorksheetName "Sheet1"
```
